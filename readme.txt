The project is about to fetch data from youtube and to store the data in unstructured as well as structured format
-------------------------------------------------------------------------------------------------------------------

Fetch data from youtube:
	1. enter the channel id it will fetch all channel data-> playlist-> video-> comments using google API
	2. for storing unstructured data mongodb is used
	3. for storing structured data MYSQL is used 

project contains 5 different pages for representing data
	1. video channel info: provides the details about video data that are related to channel
	2. engagement metrics: which provides the detail data about comments and like of videos
	3. view metrics: which provides top videos in data
	4. video duration: which provides the average duration of videos in the channel
	5. Data visualization: this page shows the trends of channel based on the year and views
